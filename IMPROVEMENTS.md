### 1. Projects & Build Tracking
**What**: Track projects you build and link to tools/materials used
- Project name, description, photos
- Link to tools and materials used in each project
- Track material consumption per project
- Project timeline and completion status
- Cost calculation (sum of materials used)
- "What can I build with what I have?" reverse lookup

### 2. Tool Maintenance System
**What**: Schedule and track tool maintenance
- Maintenance schedules (oil changes, blade replacements, calibration)
- Service history log
- Next service due date with alerts
- Maintenance cost tracking
- Links to service manuals
- Parts replacement tracking (blades, filters, belts)

**Why**: Keep tools in good condition, prevent breakdowns, extend tool life

### 3. Battery Compatibility Matrix
**What**: Track which batteries work with which tools
- Battery model tracking
- Compatible tools for each battery
- Battery charge level tracking
- Battery health/age monitoring
- "Which tools can I use right now?" based on charged batteries

**Why**: Essential for cordless tool ecosystem (Ryobi, DeWalt, Makita, etc.)

### 4. Lending Tracker
**What**: Track who borrowed what and when
- Borrower name and contact
- Item borrowed, date out, date due back
- Overdue alerts
- Return condition notes
- Borrowing history per person
- "Request return" reminders

**Why**: Stop losing tools to friends/neighbors, know who has what

### 5. Shopping List Generator
**What**: Auto-generate shopping list from multiple sources
- Items below minimum stock (already detected)
- Grouped by store (Bunnings, Mitre10, etc.)
- Estimated total cost
- Add custom items manually
- Mark items as purchased
- Export to phone for shopping

**Why**: Never forget what you need, optimize shopping trips

### 6. Warranty & Documentation Hub
**What**: Central place for all paperwork
- Warranty expiry dates with alerts
- Receipt image storage
- Manuals and documentation (PDF upload/links)
- Serial number tracking
- Claim history
- "Under warranty?" quick check

**Why**: Save money on repairs, have docs when needed

### 7. Workshop Layout/Map
**What**: Visual map of workshop showing item locations
- Drag-drop interface to place items on grid
- Color-coded by type
- Search highlights location on map
- "Where is this?" quick lookup
- Custom zones (workbench, storage, etc.)

**Why**: Find things faster, optimize workshop layout

### 8. Material Cut Calculator
**What**: Calculate cuts from stock materials
- Board footage calculator
- Cut list optimizer (minimize waste)
- Visual cut diagram
- Track remaining pieces after cuts
- Material cost per cut

**Why**: Reduce waste, plan purchases better, save money

### 9. Advanced Search & Filters
**What**: Search across all item types at once
- Global search (tools + consumables + materials + fasteners)
- Save filter presets ("My favorite tools", "Wood projects", etc.)
- Tag system for custom organization
- Smart search with fuzzy matching
- Filter by: price range, date added, location, condition

**Why**: Find anything instantly, organize your way

### 10. Data Backup & Export
**What**: Protect your data and take it anywhere
- One-click backup to file
- Export to CSV/Excel
- Import from backup
- Automatic daily backups
- Export specific categories
- Print inventory report

**Why**: Data safety, insurance claims, sharing with others
